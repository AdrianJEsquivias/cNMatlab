iv. ¿La regla del perceptrón (dadas suficientes iteraciones) siempre aprenderá a clasificar correctamente los patrones en este conjunto de entrenamiento, sin importar los pesos iniciales que usemos? Explicar.
Si, ya que el problema es linealmente separable y el algoritmo está diseñado para converger en un mismo punto pues se actualiza cada etapa de acuerdo al error obtenido; sin embargo eso no implica que se llegue al resultado más o menos rapido, pues si el valor inicial dado está muy alejado, la convergencia se dará despues de muchas iteraciones.
function [W,b] = algo_per_ite(W0,b0,p,t)
a = hardlim(W0 * p + b0);
e = t - a;
W = W0 + e * p';
b = b0 + e;
end
